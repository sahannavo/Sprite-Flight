## 🛸 About The Game
**Sprite Flight** is an action-packed 2D arcade game built to test your reflexes. Navigate your spaceship through an endless barrage of flying hexagons. One wrong move, and it's game over! This project was developed as part of the Unity Learn track to master 2D physics and game loops.

## ✨ Key Features
* 🕹️ **Smooth Controls:** Seamlessly steer your ship using mouse clicks or screen taps.
* 💥 **Dynamic Physics:** Bouncing hazards and solid screen borders that react in real-time.
* 🚨 **Collision Detection:** Pixel-perfect hitboxes that trigger explosions and game-over states.
* 💯 **Live Scoring:** An automated UI tracker that rewards you for every second you survive.

## 🎮 How to Play
1. **Launch:** Start the game and prepare for liftoff.
2. **Move:** Follow the cursor/tap to maneuver the ship.
3. **Dodge:** Avoid the incoming hexagon hazards at all costs!
4. **Survive:** The longer you stay alive, the higher your score climbs.
